# GLSL Shaders

This repository focus on various shaders using GLSL(OpenGL Shading Language)

## Building

```
mkdir build
cd build
cmake ..
```

## Demo

```
./build/myproject
```

| Key press | Event           |
|-----------|-----------------|
| Esc       | quit            |
| 1         | Phong shading   |
| 2         | Gouraud shading |
| 3         | Toon shading    |
| 4         | Edge effect     |
| 5         | Cubemap         |
